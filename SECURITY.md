# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately by opening a security advisory at:

https://github.com/geginj20/hkn-website/security/advisories

Please do **not** disclose security vulnerabilities publicly via GitHub issues.

## Response

We will acknowledge receipt of your report within 48 hours and provide an estimated timeline for a fix.

## Best Practices

- Keep dependencies up to date
- Never commit secrets, tokens, or credentials to the repository
- Use environment variables for configuration
